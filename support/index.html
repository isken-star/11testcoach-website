Help us help you

When emailing about a problem, it speeds things up enormously if you include:

  • Your app version (Parent Mode → Tools → Support → About).
  • Your device model and iOS version.
  • What you were doing when the problem happened — screenshots welcome.
  • If it's about a specific question, the question ID from the report screen.

Common questions

I forgot my Parent Mode PIN.

Email us from the address you'd like us to reply to and we'll walk you through the reset options for your situation.

How do I move my data to a new device?

Use Parent Mode → Tools → Data & Backup to create a backup file, transfer it to the new device (AirDrop or Files), then restore it from the same screen.

I found a question I think is wrong.

Brilliant — that helps everyone. Use the in-app report button on the question, or email us the question ID, and we'll review it.
